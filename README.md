# Impossible Architecture Explorer

A self-contained Three.js first-person non-Euclidean prototype.

## Run

From this folder, start any static server, for example:

```bash
python3 -m http.server 8080
```

Then open:

`http://localhost:8080`

## Controls

- WASD: move
- Mouse: look
- Shift: sprint
- Space: jump

## Features implemented

- First-person smooth movement
- Render-to-texture portal pairs with teleport crossing
- Impossible scale transition (small lobby -> huge internal room)
- Looping hallways (A/B contradictory loops)
- Gravity shift zones (sideways + inverted feeling)
- Escher-style stair wrap
- Collect 5 artifacts objective
- Contradictory minimap/path system
- Liminal/dreamcore palette, fog, and subtle horror events
- Procedural ambient drone + intermittent distant tones
- Room state mutation when turning around
